Language statistics

This website converts per word to per line rates based (among others) on the language conversion metrics provided by Wolfram Alpha.

The number of characters per word varies greatly per language, as can be seen in the table below. Wolfram Alpha calculates these values based on a statistical analysis of general texts and this might be different for texts in other areas. I therefore advise anyone to use this tool with caution.

You can verify your own results by using the Microsoft Word count tool or a CAT tool (like MemoQ).
